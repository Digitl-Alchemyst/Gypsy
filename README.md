<h3>
<ul><b>How to use</b></ul>
</h3>

## **Project Dependencies**

    Node.js 18.16.0
    npm/npx 9.5.1
    Next.js 13.4.4

## Packages

    HeroIcons: 2.0.18
    OpenAI: 3.2.1
    TailwindCSS/Typography: 0.5.9

## Install Node Modules

    npm install


## Run Application

    npm run dev
